# 每日分析报告

这里保存每天早报、晚报的原始分析报告和截图证据。

## 目录规则

- `YYYY-MM-DD-早报/`
- `YYYY-MM-DD-晚报/`

每个目录至少包含：

- 高清图文报告 PNG/JPG。
- HTML 源文件。
- 每个案例的首屏、详情中段、详情结束边界截图。
- 分析摘要或索引文件。

## 使用规则

- 这里保存“分析过程”和“判断依据”。
- 不是所有搜索到的商品都算好案例。
- 每个案例必须写优点、缺点、是否适合借鉴、适合借鉴到哪里。
- 只有真正适合用于用户产品图文制作的内容，才沉淀到 `图文制作方案`。
- 后续制作主图、详情页、短视频脚本时，优先从这里按风格、场景、产品定位匹配合适方案。
