# 2026-06-14 晚报 - 可复用图文方案

## 适合沉淀的方向

### 1. 法式立体提花下午茶路线
- 关键词：立体提花、暖色室内、下午茶、完整餐桌
- 适用：中高客单、轻奢复古、法式餐桌布
- 重点：桌椅、灯、器皿一起拍，强调完整场景关系

### 2. 美式复古蓝花圆桌路线
- 关键词：蓝花、圆桌、客厅、提花、安静克制
- 适用：家居感主图、客厅桌布、系列化视觉
- 重点：适合做稳定大货款，也适合做礼品款

### 3. 蕾丝轻柔法式路线
- 关键词：蕾丝边、浅色布面、南洋中古、轻柔
- 适用：花边款、茶几盖巾、小圆桌
- 重点：细节图要拍花边，不要只拍整体

### 4. 春夏花园提花路线
- 关键词：户外树下、满氛围桌面、提花、花园感
- 适用：春夏、节日、拍照型桌布
- 重点：适合做主视觉，不适合塞很多功能词

### 5. 小雏菊褶皱边路线
- 关键词：小雏菊、薰衣草、荷叶边、日常清新
- 适用：基础款、可爱款、轻日常桌布
- 重点：更适合做入门款和走量款

## 可直接复用的模块
- 首屏完整桌面场景
- 中段尺寸 / 材质 / 评价信息
- 尾部推荐卡流
- 纹理近景 + 规格区

## 明确要避开
- 防油、防水、防烫、防尘当主卖点
- 促销页感太强
- 只有平铺，没有场景
- 文字堆太满，版式像参数表

## 通用提示词骨架
`质量 + 主体 + 场景 + 风格 + 构图/镜头 + 光线/色彩 + 细节`

### 正向示例
- 8K高清，法式复古立体提花桌布，完整下午茶餐桌场景，暖色室内灯光，轻奢家居氛围，布料纹理清晰，商业摄影
- 8K高清，法式复古轻奢提花桌布，田园花园场景，树下自然光，满氛围桌面，商业摄影，低饱和但色彩丰富

### 反向示例
- 模糊, 低质, 畸形, 多余肢体, 文字, 水印, 丑, 变形, 色差
