# 2026-06-14 早报 - 可复用图文方案

## 适合沉淀的方向

### 1. 英式花鸟提花路线
- 关键词：鸟雀、枝叶、提花、暖色室内、完整餐桌
- 适用：用户产品主图、详情页首屏、系列主视觉
- 重点：不要只拍布面，要拍桌布与桌椅、器皿、花瓶的关系
- 可复用模块：首屏场景图 / 纹理近景 / 推荐卡流延展

### 2. 莫奈油画蓝系路线
- 关键词：蓝绿色、油画花卉、户外草地、轻艺术感
- 适用：春夏款、拍照型桌布、花园场景桌布
- 重点：道具要少而准，保留花器和桌面层次即可
- 可复用模块：户外场景主图 / 细节近景 / 规格区

### 3. 玫瑰蕾丝边路线
- 关键词：玫瑰碎花、白底、蕾丝边、轻柔清新
- 适用：文艺小清新、田园花边款、礼品型桌布
- 重点：边缘装饰要清晰，适合做细节卖点图

### 4. 复古花卉满氛围路线
- 关键词：复古花卉、提花、轻奢、满氛围桌面
- 适用：中高客单、复古轻奢、节日桌面
- 重点：整体桌面要有层次，避免只拍一块布

### 5. 日常浅绿基础款路线
- 关键词：浅绿、白椅、木地板、干净自然光
- 适用：基础款、日常款、走量款
- 重点：这条线主要学结构，不学功能话术

## 明确要避开
- 防油、防水、防烫、防尘作为首卖点
- 标题里堆太多功能词
- 参数页感太强、像低价促销页
- 只拍平铺图，不拍真实餐桌关系

## 通用提示词骨架
`质量 + 主体 + 场景 + 风格 + 构图/镜头 + 光线/色彩 + 细节`

### 正向示例
- 8K高清，英式花鸟提花桌布，完整圆桌餐桌场景，枝叶与小鸟图案，暖色室内自然光，低饱和，布料纹理清晰，商业摄影
- 8K高清，莫奈花园风圆桌桌布，蓝绿色油画花卉图案，草地户外餐桌场景，柔和自然光，低饱和，艺术感商业摄影

### 反向示例
- 模糊, 低质, 畸形, 多余肢体, 文字, 水印, 丑, 变形, 色差
