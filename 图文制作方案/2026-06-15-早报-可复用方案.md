# 2026-06-15 早报｜可复用图文制作方案

本文件只收录适合用户桌布产品继续复用的方案。功能卖点如防油污、防水、防烫不作为核心方向。

## 方案 1：油画花鸟 / 莫奈花园主视觉

- 适用范围：有明确花卉、鸟类、自然庭院感的桌布
- 可学习点：
  - 主图要先建立主题，不先讲参数
  - 颜色体系要连续，最好能形成一组系列色号
  - 详情页顺序建议：首屏主题图 -> 场景图 -> 纹理细节 -> 尺寸/规格 -> 收尾推荐
- 适合用户产品的图文方向：
  - 更偏设计感、小众感、油画感、春日庭院感
  - 适合做主图和详情页首屏
- 生图提示词骨架：
  - 质量 + 主体 + 场景 + 风格 + 构图 + 光线 + 细节
  - 示例：8K，高级感油画花鸟桌布，木质餐桌，莫奈花园色调，俯拍构图，柔光，花卉提花纹理清晰

## 方案 2：法式蕾丝边轻盈桌布

- 适用范围：白色、米色、浅金色、纱感、镂空、绣花边桌布
- 可学习点：
  - 边缘结构要拍出来，蕾丝/花边必须能被看见
  - 适合用窗边自然光、低饱和、轻阴影
  - 详情页可以加入边缘特写、局部纹理、桌面搭配
- 适合用户产品的图文方向：
  - 法式、轻复古、婚礼桌面、柜面盖布、床头柜软装
  - 适合做补充款和氛围款
- 生图提示词骨架：
  - 白色蕾丝桌布，法式轻奢，柔光，边缘镂空清晰，居家场景，细节锐利

## 方案 3：纯色麻点 / 低饱和基础款

- 适用范围：纯色、浅色、材质感明显的基础桌布
- 可学习点：
  - 不要只堆颜色和尺寸，要补材质特写
  - 适合拍垂坠感、触感、纹理近景
  - 页面结构要更克制，用空间图代替大段文字
- 适合用户产品的图文方向：
  - 作为基础款、空间补充款、低风险搭配款

## 不建议沉淀为主方案的方向

- 过度图案化但缺少场景的条纹桌布
- 纯目录式、参数堆叠式页面
- 白底过曝、只拍平面、看不出边缘结构的蕾丝桌布
