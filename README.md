# 微信小店运营

这个项目用于低频、合规地辅助运营微信小店。

当前固定方式：

- 使用已经登录的 Playwright MCP 当前标签页。
- 在同一标签页内查看微信小店后台数据。
- 默认只读，不自动提交任何表单。
- 数据入口包括交易数据、商品数据、人群数据、电商罗盘。

后续目标：

- 自动生成每日/每周运营日报。
- 根据数据给出商品、内容、直播、短视频、达人合作建议。
- 逐步增加自动上架辅助，但发布前必须人工确认。
- 逐步增加付费自动化辅助，但投放前必须人工确认预算和动作。

素材和表单约定：

- 产品原始素材统一放到 `产品素材图/产品名或款号/`
- 单品信息统一填到 `产品信息表/产品名或款号.md`
- 图文可复用方案放到 `图文制作方案/`
- 每日分析原始报告放到 `每日分析报告/`

重要约束：

- 不绕过平台限流、登录、验证码、权限或风控。
- 不做无人值守的资金、售后、发货、投放、改价、发布操作。
- 默认低频、只读、人工确认后执行。
