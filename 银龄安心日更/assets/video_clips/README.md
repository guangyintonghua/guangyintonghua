# 动态素材库

本目录只存放用于“银龄安心日更”的真实动态视频素材。

## 目录规则

- `mixkit/`：来自 Mixkit 的可商用动态素材。
- `shot/`：你自己拍摄的真实素材。
- `topic/`：按内容主题拆分的素材池，例如 `health_diet_breakfast`、`anti_fraud_phone`。

## 使用规则

- 只放真实动态视频，不放静态图片。
- 每个主题目录至少准备 4 段可切换镜头。
- 单段素材建议 3-15 秒，横屏优先。
- 素材文件不提交到 GitHub，只保留目录结构和说明文件。
