# 银龄安心日更

面向 50-65 岁中老年人的实用安全内容日更项目。

主平台：今日头条。

核心目标：每天产出可发布的内容成品，用于获取平台内容收益。暂不考虑卖货、商单、私域转化。

## 每日交付

只交付可直接上传的成品文件：

- `今日头条文章.md`：1 篇可直接发布的今日头条文章，1200-1800 字。
- `今日头条横屏视频.mp4`：1 条 16:9 横屏短视频成品，1-3 分钟，必须基于真实动态镜头，不接受静态图片轮播成片。

资料核验、脚本、字幕、封面、配图提示词和自检只作为内部生产流程，不单独输出给发布者。

## 内容栏目

- 社保医保：养老金认证、医保报销、退休手续、补贴政策。
- 手机反诈：微信设置、支付安全、陌生链接、医保卡骗局。
- 生活避坑：保健品、消费陷阱、退休再就业、合同借钱风险。
- 家庭养老：子女沟通、养老安排、独居安全、财产边界。
- 健康饮食：三餐搭配、控盐控油控糖、蛋白质、粗粮、买菜避坑、饮食误区。

## 健康饮食边界

只做日常膳食科普，不做疾病治疗、药物建议或医疗承诺。

优先使用《中国居民膳食指南（2022）》《中国老年人膳食指南（2022）》、国家卫健委、中国疾控中心、健康中国、人民网、新华网、公立医院公开科普等来源。

## 风格要求

- 中国化表达，使用中国家庭、饭桌、菜市场、厨房、社区等生活场景。
- 口语化，面向中老年读者和观众。
- 可用中国俗语、老话、成语，但不硬凑。
- 避免明显模板句和低质 AI 感。
- 数据、政策、健康建议必须可追溯，不做虚假阐述。

## 自动化时间

每日中国时间 09:00 产出当天内容包。
